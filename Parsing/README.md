# Парсинг данных о квартирах
## Извлекаемые данные
- физический адрес
- количество комнат
- площадь квартиры
- количество этажей в доме
- этаж квартиры
- цена
- ссылка на объявление
- текст объявления
- ремонт
- санузел
- вид из окон
- год постройки
- лифт
- мусоропровод
- тип дома
- парковка
## Сохранение
Ввиду небольшого размера выборки (порядка нескольких тысяч), данные сохраняются в csv-файл.
